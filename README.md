# Automating-Employee-Data
The objective of this scenario is to automate the import of employee data, display department manager information directly on incidents, and set a custom priority default for incidents to improve data accuracy, provide quick access to relevant information, and streamline the incident resolution process in ServiceNow. demovideo link: https://drive.google.com/file/d/1nF-XAI4vShwXg3S2ilaozV6wsuN8WgIF/view?usp=sharing
